# Contributing Guide

Thank you for your interest in MyTypist! This repository contains public product documentation and branding assets.

## How to Contribute

### Reporting Issues
- Open a GitHub issue describing the problem or suggestion.
- Include screenshots or references where applicable.

### Pull Requests
1. Fork the repository.
2. Create a feature branch (`feature/your-change`).
3. Commit your changes with a clear message.
4. Submit a PR against the `main` branch.

### What We Accept
- Documentation fixes and improvements
- Screenshot/image asset updates
- Roadmap suggestions via issues

Application code lives in private repositories. For contributions to the backend or frontend, reach out to the maintainers.

Thank you
